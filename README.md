# Principle
- 最小化软件需求。 by Rin
- 将软件需求拆分到桌面端和移动端。 by Rin
- 避免依赖原生软件以实现通用性。 by cyancat
- 所有的手机都是电子垃圾，只做原生功能替换和尽可能少的额外需求补充。 by Rin
